# Vtagu_API (Nest + MySQL)

Quick start:

1. Ensure Node.js (>=16) is installed.
2. Install dependencies (already done if you followed the setup):

```powershell
npm install
```

3. Create a MySQL database and update `.env` with credentials.

4. Run in development mode:

```powershell
npm run start:dev
```

API endpoints:

- `GET /api/posters` — list poster
- `GET /api/movies/trending` — list trending movies
